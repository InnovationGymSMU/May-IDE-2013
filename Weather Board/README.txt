Sparkfun Weather Board V3
-----------------------------
Firmware has been modified to fit the needs of this IDE.

Output is asterisk separated list of the format:
	1. Temperature in degrees fahrenheit
	2. Percent Humidity 
	3. Dewpoint in degrees Fahrenheit
	4. Barometric Pressure in Inches of Mercury

Before use, make sure the Comm Switch is set to USB and the Power Switch is set to USB.
